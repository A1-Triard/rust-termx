# termx-screen-ncurses

ECS-based TUI with XAML. Intended to abstract over ncurses/notcurses/win console API/DOS text mode/etc.

Basic text screen implementation for Unix/Linux platform.
Use [termx-screen](https://crates.io/crates/termx-screen) as platform-independent frontend.
